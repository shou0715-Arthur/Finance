# DCF估值教育系統 規格說明書

> **用途說明**: 本規格說明書作為AI提示語使用，將此完整內容提交給AI，AI將根據規格生成完整的應用程式碼。

## 📋 系統概述

### 系統名稱
【Code Gym】DCF估值教育系統

### 核心功能描述
建立一個基於網頁的DCF（現金流折現）估值教育工具，能夠：
1. 自動載入公司實際財務數據作為DCF模型的基礎參數
2. 提供互動式參數調整功能，讓學員理解不同假設對估值的影響
3. 執行專業級DCF估值計算，包括企業價值、股權價值和每股價值
4. 提供多層次的分析結果展示，包括敏感性分析和情境分析
5. 整合AI分析功能，提供專業的估值解讀和投資教育洞察

### 技術架構要求
- **界面框架**: 使用 Streamlit 框架
- **數據來源**: Financial Modeling Prep (FMP) Custom DCF API
- **AI 模型**: OpenAI GPT-5系列
- **AI 模型版本**: gpt-5-mini
- **視覺化工具**: 互動式圖表 (使用 Plotly Graph Objects)
- **數據處理**: Pandas, NumPy
- **HTTP請求**: requests
- **日期處理**: datetime
- **部署方式**: 可直接在瀏覽器中運行

### 重要套件版本要求
- **OpenAI套件**: 使用 openai 版本 1.0.0 或以上的新API格式
- **禁用舊版API**: 絕對不可使用 ChatCompletion.create 舊格式
- **必須使用新格式**: 使用 OpenAI 客戶端初始化方式和 chat.completions.create 方法

### F-002: 數據獲取功能
**功能目標**: 雙階段DCF計算流程，先載入公司實際數據，再進行自定義計算

**第一階段：公司基礎數據載入**
- **API端點**: https://financialmodelingprep.com/stable/custom-discounted-cash-flow
- **請求參數**: symbol={ticker}&apikey={api_key}
- **數據處理**: 提取公司實際財務參數作為調整基準
- **參數提取清單**:
  - revenuePercentage: 營收成長率（轉換為小數）
  - ebitdaPercentage: EBITDA利潤率（轉換為小數）
  - ebitPercentage: EBIT利潤率（轉換為小數）
  - capitalExpenditurePercentage: 資本支出比例（轉換為小數，取絕對值）
  - depreciationPercentage: 折舊攤銷比例（轉換為小數）
  - operatingCashFlowPercentage: 營運現金流比例（計算或預設）
  - 資產負債表相關比例（現金、應收、存貨、應付等）
  - 稅率、成長率等財務參數
  - 資本成本相關參數（WACC、股權成本、債務成本等）

**第二階段：自定義DCF計算**
- **API端點**: 同第一階段，但傳送調整後的參數
- **參數轉換規則**:
  - 百分比參數：需轉換為API要求格式（部分需乘以100，部分保持小數）
  - 風險參數：riskFreeRate、marketRiskPremium、costOfDebt等需要百分比格式
  - 成長和利潤參數：保持小數格式
- **計算結果**: 完整的DCF估值結果，包含所有中間計算步驟

### F-003: 參數調整功能
**調整界面設計**: 基於公司實際數據提供合理的調整範圍和預設值

**基礎參數組**:
- **營收成長率滑桿**: 範圍-10%到200%，步進0.5%，顯示公司預設值對比
- **EBITDA利潤率滑桿**: 範圍5%到50%，步進1%，顯示公司預設值對比
- **資本支出比例滑桿**: 範圍2%到20%，步進0.5%，顯示公司預設值對比

**折現率參數組**:
- **股權成本滑桿**: 範圍5%到20%，步進0.1%，附帶CAPM計算說明
- **債務成本滑桿**: 範圍1%到10%，步進0.1%，附帶稅前成本說明
- **無風險利率滑桿**: 範圍2%到8%，步進0.1%，附帶10年期國債參考
- **市場風險溢價滑桿**: 範圍4%到10%，步進0.1%，附帶股市風險溢價說明
- **Beta係數滑桿**: 範圍0.5到2.5，步進0.1，附帶系統風險係數說明

**進階參數摺疊區域**:
- **永續成長率滑桿**: 範圍1%到5%，步進0.1%，附帶長期成長率說明
- **有效稅率滑桿**: 範圍15%到35%，步進1%，附帶公司實際稅率說明
- **營運現金流比例滑桿**: 範圍5%到40%，步進1%，附帶現金流品質說明

**參數互動功能**:
- 每個參數都附帶help提示，顯示公司實際數值
- 重置按鈕可恢復為公司原始數據
- 即時顯示參數調整對比表

### F-004: DCF計算與驗證功能
**計算觸發機制**: 用戶點擊"計算DCF估值"按鈕後執行完整計算流程

**計算驗證要求**:
- **合理性檢查**: DCF價格應在10-10000美元範圍內
- **市價差離度檢查**: 與當前股價的差離程度評估
- **企業價值一致性檢查**: 現金流現值+終值=企業價值
- **終值佔比檢查**: 終值應佔企業價值50-90%
- **計算變化分析**: 對比原始DCF與調整後DCF的變化

**錯誤處理機制**:
- API請求失敗的友善提示
- 參數異常值的警告提示
- 計算結果異常的合理性提醒
- 網路連接問題的重試建議

### F-005: 結果展示功能
**主要指標展示**: 使用4欄指標元件展示核心結果
- **DCF每股價值**: 顯示計算後的內在價值

- **估值差異**: 顯示被低估/高估的百分比和趨勢

**參數對比展示**: 使用雙欄表格對比個人設定vs公司實際數據
- **左欄**: 基礎參數對比表（營收成長率、EBITDA率、資本支出、股權成本等）
- **右欄**: 進階參數對比表（永續成長率、稅率、營運現金流比例等）

**計算變化展示**: 使用3欄指標顯示

**WACC計算明細表**: 加權平均資本成本組成

- 影響程度排序：永續成長率 > WACC > 營收成長率

**風險提醒資訊**: 敏感性分析重點
- 永續成長率對估值影響最大（±35%）
- WACC變化對估值顯著影響（±25%）
- 營收成長率需要基本面支撐
- 建議進行多情境分析，避免過度依賴單一假設
